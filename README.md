# 🚢 Titanic Survival Analysis (Exploratory Data Analysis)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the famous Titanic dataset.  
The Titanic disaster is one of the most well-known maritime tragedies in history, and analyzing the dataset helps us understand patterns related to passenger survival.

Using Python and data visualization techniques, this project explores how factors such as **gender, age, passenger class, and fare** influenced survival chances.



## 🎯 Objectives
The main objectives of this project are:

- Explore the Titanic dataset and understand its structure
- Perform **data cleaning and preprocessing**
- Identify factors that influenced passenger survival
- Visualize patterns using graphs and charts
- Extract meaningful insights from the dataset



## 📊 Dataset Description

The Titanic dataset contains information about passengers aboard the Titanic. Important columns include:

| Column Name | Description |
|-------------|-------------|
| PassengerId | Unique ID of each passenger |
| Survived | Survival status (0 = No, 1 = Yes) |
| Pclass | Passenger class (1st, 2nd, 3rd) |
| Name | Passenger name |
| Sex | Gender of passenger |
| Age | Age of passenger |
| SibSp | Number of siblings/spouses aboard |
| Parch | Number of parents/children aboard |
| Ticket | Ticket number |
| Fare | Ticket price |
| Cabin | Cabin number |
| Embarked | Port where passenger boarded |


## 🛠️ Technologies Used

The following tools and libraries were used in this project:

- **Python**
- **Pandas** – Data manipulation and analysis
- **NumPy** – Numerical operations
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical data visualization
- **Jupyter Notebook**



## 📈 Analysis Performed

### 1️⃣ Survival Distribution
Analysis of how many passengers survived vs how many did not survive.

### 2️⃣ Survival by Gender
Comparison of survival rates between **male and female passengers**.

### 3️⃣ Survival by Passenger Class
Analysis of survival chances based on **ticket class (1st, 2nd, 3rd)**.

### 4️⃣ Age Distribution
Visualization of passenger age distribution to understand the age range of passengers on the Titanic.

### 5️⃣ Correlation Analysis
A **correlation heatmap** was created to analyze relationships between numerical features.



## 🔍 Key Insights

- Female passengers had a **higher survival rate** compared to male passengers.
- Passengers traveling in **first class had better survival chances** than those in lower classes.
- Age and ticket fare also showed some relationship with survival probability.
- Social and economic factors played an important role in survival.



## ✅ Conclusion
This project demonstrates how **Exploratory Data Analysis (EDA)** can be used to uncover patterns and insights in historical datasets. By analyzing the Titanic dataset, we gain a better understanding of the factors that influenced survival during the disaster.
